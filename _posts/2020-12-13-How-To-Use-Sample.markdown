# 예제 돌려보는 방법

많은 기사와 튜토리얼들은 샘플 코드에 대한 링크가 포함되어 있다.

1. asp.net repository zip file을 다운로드한다.
2. Docs-master.zip파일을 압축을 푼다.
3. 샘플 디렉토리로 가는 것을 돕기위해 샘플 링크에 있는 url을 사용한다.
